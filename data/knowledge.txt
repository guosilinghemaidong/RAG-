========================================
星云科技有限公司员工手册
========================================

第一章 公司概况

1.1 公司简介
星云科技有限公司成立于2018年，总部位于深圳南山区科技园，是一家专注于人工智能和云计算解决方案的高科技企业。公司目前拥有员工500余人，其中研发人员占比超过60%。公司核心产品包括：星云智能客服系统、星云数据中台、星云AI开发平台。

1.2 公司文化
公司秉承"让AI赋能每一家企业"的愿景，倡导开放、创新、协作的工作氛围。公司实行扁平化管理，鼓励跨部门协作和技术分享。每周五下午设有"星云技术沙龙"，员工可自愿参加技术分享和讨论。

1.3 组织架构
公司设有以下主要部门：
- 研发中心：负责核心产品开发和技术创新，下设AI算法组、平台架构组、前端体验组
- 产品部：负责产品规划、需求分析和用户体验设计
- 市场部：负责品牌推广、客户拓展和行业合作
- 人力资源部：负责招聘、培训、绩效考核和员工关系
- 财务部：负责财务核算、预算管理和资金运营
- 运维部：负责系统运维、安全保障和技术支持

第二章 薪酬与福利

2.1 薪酬体系
公司采用"基本工资+绩效奖金+项目奖金"的薪酬结构：
- 基本工资：根据岗位级别确定，每年4月进行一次薪酬回顾和调整
- 绩效奖金：每季度根据绩效考核结果发放，考核等级分为S（卓越）、A（优秀）、B（良好）、C（达标）、D（待改进），对应奖金系数分别为2.0、1.5、1.0、0.5、0
- 项目奖金：参与重点项目的员工，项目上线后根据贡献度发放额外奖金

2.2 五险一金
公司依法为员工缴纳五险一金：
- 社保：养老保险、医疗保险、失业保险、工伤保险、生育保险
- 公积金：按12%比例缴纳，公司与个人各承担一半

2.3 补充福利
除法定福利外，公司还提供以下补充福利：
- 商业医疗保险：覆盖员工及配偶、子女，年度保额50万元
- 年度体检：每年一次，涵盖常规检查和专项筛查
- 餐补：每日30元，按月随工资发放
- 交通补贴：每月500元
- 通讯补贴：每月200元
- 节日福利：春节、中秋、端午等传统节日发放礼品或购物卡
- 团建活动：每季度一次部门团建，年度预算人均800元

2.4 休假制度
- 年假：入职满1年后享有5天年假，满3年10天，满5年15天，满10年20天
- 病假：每年累计不超过10天，超出部分按事假处理
- 事假：需提前申请，最长不超过15天/年
- 产假/陪产假：产假178天，陪产假15天
- 婚假：3天
- 丧假：3天（直系亲属）

第三章 办公规范

3.1 工作时间
公司实行弹性工作制：
- 核心工作时间：10:00-16:00（必须在场）
- 弹性时间：可在8:00-10:00之间到岗，16:00-20:00之间离岗
- 每日工作时长不少于8小时
- 每周工作5天，周末双休

3.2 办公环境
- 办公区域禁止吸烟
- 会议结束后需整理会议室，恢复桌椅摆放
- 公共区域（茶水间、休息区）使用后需自行清理
- 办公设备（投影仪、白板等）使用后需归位

3.3 信息化管理
- 公司邮箱：格式为姓名@xingyun-tech.com，禁止用于私人事务
- 企业微信：所有工作沟通必须使用企业微信，禁止使用个人微信讨论工作
- VPN：远程办公需通过公司VPN接入内网
- 文件共享：使用公司NAS或飞书文档，禁止使用第三方云盘存储工作文件

第四章 技术开发规范

4.1 代码管理
- 所有项目代码必须托管在公司GitLab仓库
- 分支策略：master（生产）、develop（开发）、feature/*（功能）、hotfix/*（紧急修复）
- 代码合并必须经过Code Review，至少1名审核人通过才能合并
- 每次提交必须写明commit message，格式：type(scope): description

4.2 开发流程
- 需求评审 → 报告设计 → 开发 → 自测 → Code Review → 测试 → 上线
- 每个功能开发周期不超过2周
- 上线前必须通过完整的回归测试

4.3 技术选型（当前主流技术栈）
- 后端：Python（FastAPI）+ Java（Spring Boot）
- 前端：React + TypeScript
- AI框架：LangChain + LlamaIndex
- 向量数据库：Milvus
- 容器化：Docker + Kubernetes
- 监控：Prometheus + Grafana

4.4 安全规范
- 所有API接口必须经过鉴权
- 数据库密码、API密钥等敏感信息禁止硬编码，统一使用环境变量或配置中心
- 生产数据禁止在开发环境使用
- 每季度进行一次安全审计

第五章 退款与投诉处理流程

5.1 客户退款流程
退款申请处理分为以下步骤：
1. 客户提交退款申请：通过星云智能客服系统或邮件提交退款申请，需提供订单号和退款原因
2. 初审：客服人员收到申请后48小时内进行初审，核实订单信息和退款原因
3. 审核确认：
   - 退款金额≤5000元：由客服主管直接审批
   - 退款金额5000-50000元：由区域经理审批
   - 退款金额>50000元：由财务总监审批
4. 退款执行：审批通过后，财务部在3个工作日内完成退款操作
5. 结果通知：退款完成后，系统自动发送通知邮件给客户

5.2 异常处理
- 退款代码503：表示退款系统暂时不可用，客服应告知客户稍后重试，并记录异常日志
- 退款代码505：表示订单信息校验失败，需客户重新提交正确的订单号
- 退款代码404：表示订单不存在，需核实客户是否为本公司客户

5.3 投诉处理
- 一级投诉（服务态度）：24小时内响应，3个工作日内处理完毕
- 二级投诉（产品质量）：48小时内响应，5个工作日内处理完毕
- 三级投诉（重大损失）：立即响应，成立专项小组处理，7个工作日内出具处理方案

5.4 客户满意度追踪
退款和投诉处理后，客服团队需在7天内进行客户回访，确认问题是否得到妥善解决。客户满意度评分低于3分（满分5分）的案例，需上报管理层进行复盘。
